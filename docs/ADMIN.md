# دليل لوحة التحكم — مسارات المستكشف

لوحة تحكم لإدارة كل محتوى الموقع. أي تعديل يظهر على الموقع فورًا.

## الدخول

- الرابط: `/admin` (يحوّلك إلى `/admin/login` إن لم تكن مسجّلًا).
- الحساب الأول يُنشأ تلقائيًا عبر `npm run db:seed` من متغيّرات `ADMIN_EMAIL` /
  `ADMIN_PASSWORD`. **غيّر كلمة المرور بعد أول دخول** من «المستخدمون».

## الأدوار (الصلاحيات)

| الدور | يستطيع |
|------|--------|
| **مدير (admin)** | كل شيء: المحتوى + الإعدادات + المستخدمون + سجل التدقيق |
| **محرّر (editor)** | المحتوى فقط (لا يرى الإعدادات/المستخدمين/التدقيق) |

## الأقسام

### لوحة المعلومات
نظرة عامة بأعداد كل نوع محتوى، وعدد الاستفسارات غير المقروءة.

### الاستفسارات (رسائل نموذج التواصل)
- كل رسالة من نموذج «تواصل معنا» تُحفظ هنا تلقائيًا (حتى لو تعطّل البريد).
- **بحث** بالاسم/البريد/النص، و**فلترة** حسب الحالة، و**تصدير CSV**.
- فتح الرسالة يعلّمها «مقروءة»، ويتيح: **رد عبر البريد**، تعليم كغير مقروء، حذف.
- **الحالة**: جديد ← قيد المعالجة ← مغلق (لمتابعة المبيعات).

### المحتوى (إضافة/تعديل/حذف/ترتيب)
- **الخدمات / المنتجات**: لكل عنصر معرّف (slug) يظهر في الرابط، أيقونة
  (اسم Material Symbol مثل `router`)، ومزايا (سطر لكل ميزة).
- **الأعمال**: الجهة، وصف العمل، التصنيف، والحالة (قائم/منتهي).
- **الشركاء**: الاسم، القطاع، أيقونة أو **شعار مرفوع**.
- **الأسئلة الشائعة**، **المزايا** (لماذا مسارات)، **القطاعات**، **مراحل العمل**:
  أيقونة + عنوان + وصف + ترتيب.
- حقل **الترتيب** يتحكّم بترتيب الظهور (الأصغر أولًا).

### الصفحات (منشئ الصفحات)
- أنشئ صفحات جديدة بمعرّف خاص تظهر على `/<المعرّف>`.
- المحتوى **بلوكات**: غلاف، عنوان، فقرة، قائمة، صورة، دعوة لإجراء — أضف/رتّب/احذف.
- زر **نشر** يتحكّم بظهور الصفحة للزوّار (المسودّات مخفية).

### الإعدادات (للمدير فقط)
- **بيانات الشركة**: الاسم، الوصف، الرابط، البريد، الهاتف، واتساب، العنوان.
- **الشعار**: ارفع شعارًا (وآخر أبيض للفوتر الداكن) لتجاوز الشعار الافتراضي.
- **استقبال الاستفسارات**:
  - بريد الاستقبال (يقبل أكثر من بريد مفصولة بفاصلة).
  - **Resend**: أنشئ حسابًا على [resend.com]، وثّق نطاقك، وأدخل المفتاح وعنوان
    المُرسِل ليصلك نسخة بالبريد. (بدونه تبقى الرسائل محفوظة في «الاستفسارات».)
- **الحماية من السبام (Cloudflare Turnstile)**: أنشئ موقعًا في Turnstile وأدخل
  Site Key + Secret Key. تظهر أداة تحقّق في النموذج وتُرفض محاولات الروبوت.
  اتركهما فارغين للتعطيل.
- **التنبيهات (Telegram)**: أنشئ بوت عبر `@BotFather`، واحصل على `Chat ID`، وأدخل
  Token + Chat ID ليصلك تنبيه فوري بكل استفسار.
- المفاتيح السرّية لا تُعرض بعد حفظها؛ اترك الحقل فارغًا للإبقاء على القيمة الحالية.

### المستخدمون (للمدير فقط)
- إضافة/تعديل مستخدمين وتحديد الدور. كلمات المرور مشفّرة.
- لا يمكنك حذف حسابك، ولا حذف آخر مدير.

### سجل التدقيق (للمدير فقط)
عرض آخر العمليات (إضافة/تعديل/حذف) ومن قام بها ومتى.

## رفع الصور
- مسموح: JPG, PNG, WEBP, GIF, AVIF (حتى 5 ميغابايت). تُحفظ في `public/uploads`.
- ملفات SVG غير مسموحة في الرفع لأسباب أمنية؛ ضع شعارات SVG في `public/` مباشرة.

## النسخ الاحتياطي
شغّل `./scripts/backup.sh` أو اجدوله عبر cron (انظر `DEPLOY.md`).
