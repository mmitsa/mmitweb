# شعارات العلامة — Brand assets

ضع ملفات الشعار الرسمية هنا **بنفس هذه الأسماء بالضبط** وسيستخدمها الموقع كما هي
دون أي تعديل برمجي:

| الملف | الاستخدام | الوصف المطلوب |
|------|-----------|----------------|
| `logo-mark.svg` | الهيدر + شبكة الهيرو (خلفية فاتحة) | علامة الـ«M» بالألوان (أزرق + المثلث الأسود)، خلفية شفافة |
| `logo-mark-white.svg` | الفوتر (خلفية كحلية) | علامة الـ«M» بالأبيض، خلفية شفافة |
| `logo-full.svg` | اختياري — اللوك الكامل (شعار + نص عربي) | خلفية شفافة |
| `icon.png` | الأيقونة المفضّلة (favicon) 512×512 | خلفية شفافة أو بيضاء |

> الملفات الحالية هنا **مؤقتة (stand-in)** فقط لمنع كسر الموقع — استبدلها
> بالملفات الرسمية المُصدّرة (SVG مفضّل) بنفس الأسماء، ولا حاجة لأي تعديل في الكود.

التصدير من ملف العلامة: صدّر علامة الـ«M» وحدها (بدون النص) كـ SVG بخلفية شفافة.
